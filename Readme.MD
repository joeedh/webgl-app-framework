## Checkout

<pre>

git clone https://github.com/joeedh/webgl-app-framework.git
cd webgl-app-framework
bash tools/git_pull.sh

</pre>

## Building

* Build: `pnpm build`
* Watch build: `pnpm watch`
* Typecheck: `pnpm typecheck`
* Start web server: `pnpm serv`
